```

```

### "Director's Cut" Series

```
Show the "professional" version first
Then show what it took to get there
"23 takes and 4 meltdowns later..."
```

---

### "Things We Said During Filming"

```
Text-on-screen format with audio:
- "Why is my face doing that"
- "Can we take a break I'm hungry"
- "STOP MAKING ME LAUGH"
- "That was perfect- wait it wasn't recording"
- "I forgot what app we're promoting"
```

---

### "The Blooper That Became the Final Take"

```
Show a "mistake" that was actually funnier
Than what was scripted
"We kept this one."
```

## üí° PRO TIPS

1. **Never stop recording** - The best bloopers happen when you think you're done
2. **React genuinely** - Forced laughter looks fake
3. **Include everyone** - Rotate who's the "victim"
4. **Save EVERYTHING** - Even boring moments can be gold later
5. **Get consent** - Make sure everyone's okay being in bloopers
